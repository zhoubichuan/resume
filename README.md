# 个人简历

**技术栈：** Vue2.x + Vue-cli3.x + ElementUi + TypeScript + Scss

- 安装依赖

```
cnpm install
```

- 运行

```
npm run dev
```

- 打包项目

```
npm run build
```

- 单元测试

```
npm run test:unit
```

- 端到端测试

```
npm run test:e2e
```

- 代码检查

```
npm run lint
```
